# WinOptim 简介

快速调整系统设置，解决系统安装后手工逐项调整设置的繁琐，调整原则：

* 以调整 UI 展示类设置为主
* 卸载不用的自带软件
* 不做“性能”方面的调优

# 下载

从以下链接下载最新的版本：`WinOptim_版本号.zip`

> https://github.com/dsx42/WinOptim/releases

# 如何使用本工具？

解压下载好的压缩包，双击解压后的 `WinOptim.cmd` 即可
